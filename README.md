# 2018 Numerical Method

## Project Description
This project implements various numerical methods that are essential for solving mathematical problems computationally. These methods include numerical analysis techniques widely used in engineering and scientific computing. 

## Features
- Implements algorithms for root finding (e.g., Bisection Method, Newton's Method)
- Includes techniques for numerical integration (e.g., Trapezoidal Rule, Simpson's Rule)
- Offers methods for solving ordinary differential equations (ODEs)
- Provides methods for interpolation and polynomial approximation
